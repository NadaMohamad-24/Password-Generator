# 🔐 Secure Password Generator

A modern Password Generator built with Python and CustomTkinter.

## ✨ Features

- Generate strong passwords
- Password strength checker
- Copy password to clipboard
- Password history
- Clear history
- Modern Dark UI
- Light/Dark Theme

## 🛠 Technologies

- Python
- CustomTkinter
- JSON
